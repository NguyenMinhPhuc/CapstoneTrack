# **App Name**: CapstoneTrack

## Core Features:

- Student Registration: Allow students to register and provide necessary information for internship/graduation.
- Approval Workflow: Implement an approval process for student applications.
- Supervisor Assignment: Enable admins to assign faculty supervisors to students.
- Progress Tracking: Track student progress through milestones like proposal submission.
- Weekly Progress Reports: Students submit weekly progress reports.
- Final Report Submission: Students submit their final reports upon completion.
- Admin Dashboard: Provide admins with statistical overviews and system parameter settings.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to represent trust and reliability in academic tracking.
- Background color: Light gray (#ECEFF1), providing a clean, non-distracting backdrop.
- Accent color: Amber (#FFC107) to highlight important actions and notifications, analogous to blue but distinct in tone.
- Body font: 'Inter', a grotesque-style sans-serif, suitable for body text.
- Headline font: 'Space Grotesk', a proportional sans-serif for headlines.
- Use consistent, professional icons from a library like Material Icons to represent different stages and actions.
- Maintain a clear, hierarchical layout with sufficient white space to reduce cognitive load.
- Use subtle transitions and animations to provide feedback and guide users through processes.
